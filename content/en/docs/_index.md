Quality Assurance DAO (QA-DAO) is an ongoing open source project that provides support for the Cardano Project Catalyst Community. 
QA-DAO is part of the [Ryuuki](https://cardano-community-builders.github.io/) network a grouping of x projects including Catalsyt Swarm, WADA, Lauris | EU01 StakePool , Gimbalabs, Catalyst School, Eastern Town Hall ...
