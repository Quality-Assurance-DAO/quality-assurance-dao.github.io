Quality Assurance DAO (QADAO) is an ongoing open source project that provides support for the Cardano Project Catalyst Community. 

> QA-DAO is part of the [Ryuuki](https://cardano-community-builders.github.io/) stakepool network, (Twitter : [RyuukiNetwork](https://twitter.com/RyuukiNetwork)), which is a grouping of 15 projects including :
> Catalyst Swarm, [WADA](https://wadalliance.org/), [Lauris | EU01 StakePool](https://www.stakepool247.eu/), [Gimbalabs](https://www.gimbalabs.com/gimbalgrid), Catalyst School, [Eastern Town Hall](https://easterntownhall.com/), [Innovatio](https://www.innovatiofounder.com/), [AIM](https://twitter.com/AimCardano), PACE,  ...
